# NexSpy Support

Thanks for using NexSpy.

## Contact

For support, bug reports, or feature requests, please open an issue in the GitHub repository where this app is published.

## App Store Support URL

After this repository is public on GitHub, use the public URL for this file as the App Store Connect support URL:

```text
https://github.com/hrakhshani/nex-spy-game
```

## Troubleshooting

- Make sure all players have seen only their own role.
- Start a new game if the role reveal flow was interrupted.
- Restart the app if the timer or screen state does not look correct.

## Privacy

NexSpy is designed as a local pass-and-play game. It does not require an account to play.
